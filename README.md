Wordly Dictionary SPA

-> It is a single page application that allows users to search for english words, view definitions, pronunciations, synonyms and save their favourite words using local srorage.

# Features.

Search for words using a free dictionary API
View meanings and parts of speech
If audio is available you can listen to the pronunciation of it
Error handling is capable for missing and invalid words

## Technologies used.

HTML
CSS
JavaScript
Free dictionary API


### How it works

Users enters a word in the search output
JavaScript sends a request to the dictionary API
The app then displays:
 1.The word
 2.Phonetic spelling
 3.Audio pronunciation
 4.Definitions
 5.Examples

Users can also click synonyms to search instantly, save words to favourites and remove searched words

#### Cool features in this app

Favourite words are stored using localStorage
Data is still there even after refreshing the page
Users can add or remove words anytime


##### Limitations of the app

It only supports english words
Some api responses may not include Audio examples or synonyms
